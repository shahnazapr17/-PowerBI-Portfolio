# 🏥 Hospital ER Analytics Dashboard

## Overview
An emergency room analytics dashboard built in Power BI 
analysing 3,000 patient visits including wait times, 
satisfaction scores and demographic breakdowns.

## Key Metrics
- Total Patients: 3,000
- Average Wait Time: 35.35 minutes
- Average Satisfaction: 4.99/10
- Average Patient Age: 39.94

## Key Insights
- Wait time of 35 mins exceeds recommended 20 min target
- Low satisfaction score directly linked to high wait times
- 18-34 age group are the highest ER users
- White and African American patients represent highest volume

## Tools Used
- Power BI Desktop
- DAX (COUNTROWS, AVERAGE)
- Power Query
- Calculated Columns (Age Group segmentation)

## Visuals Included
- KPI Cards
- Bar Chart (Patients by Department)
- Donut Chart (Gender split)
- Line Chart (Wait Time trend)
- Bar Chart (Patients by Race)
- Donut Chart (Patients by Age Group)
- Bar Chart (Satisfaction by Department)
- Gender Slicer
