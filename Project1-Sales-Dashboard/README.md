# 🛒 Superstore Sales Dashboard

## Overview
An executive-level sales analytics dashboard built in Power BI analysing 25K orders and 13M total sales across multiple countries and product categories.

## Key Metrics
- Total Sales: 13M
- Total Profit: 1.47M
- Total Orders: 25K
- Profit Margin: 11.62%

## Key Insights
- Technology drives 45% of total profit
- Sales grew consistently year on year from 2011 to 2014
- North America is the highest revenue region

## Tools Used
- Power BI Desktop
- DAX (SUM, DIVIDE, DISTINCTCOUNT)
- Power Query
- Star Schema Data Modelling

## Visuals Included
- KPI Cards
- Bar Chart (Sales by Category)
- World Map (Sales by Country)
- Line Chart (Sales Trend)
- Donut Chart (Profit by Category)
- Year Slicer
